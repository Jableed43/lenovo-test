## Lenovo Challenge

***You will find:***
  - A lenovo responsive landing page, desktop, tablet and mobile versions
  - 6 javascript code challenges

***Technologies:***
  - Html 
  - Js
  - Css

**Steps to run the project.:**

***In visual studio code terminal write the following commands:***
   - Clone project to local repository "git clone https://github.com/Jableed43/lenovo-test.git"
   - Inside "js-challenge" directory you will find every challenge with it's own test code to run
   - Inside "landing-page" directory "npm i" to install the necessary dependencies, then "npm start" to start project
